REFLECTION

After completing this website, along with the other assignments in this module, I feel very happy and accomplished with the work I have done. Throughout this module, I have gained a much better understanding of both the technical and creative aspects of website design, which has been very rewarding. While doing this module, I learned new skills such as website coding using HTML and CSS, as well as debugging errors within my code when things did not work as expected. Learning how to identify and fix these issues helped me to become more patient and confident when working with code, as I now understand that problem-solving is a key part of the development process rather than something to be avoided.

In addition to coding, I also learned about optimal grid layouts that are commonly used in professional website design. Understanding how grids work helped me to structure my pages in a more organised and visually balanced way. This knowledge allowed me to create layouts that are easier to navigate and more visually appealing for the user. Applying these layout principles made a noticeable improvement in the overall quality of my website compared to my earlier work.

I also really enjoyed learning how to use Figma as part of this project. Planning out my website layout in Figma before implementing it into my code made the entire process much more streamlined and efficient. By designing the structure and visual elements beforehand, I was able to clearly see how each page would look and function before starting to code. This reduced the amount of trial and error and helped me to work more confidently when translating my designs into HTML and CSS.

The graphic composition project further supported my learning and development during this module. It helped me to better understand how to use the tools available in Figma, such as layers, typography controls, colour palettes, and alignment tools. Creating graphical pieces alongside the website also allowed me to think more carefully about visual hierarchy, consistency, and overall design aesthetics. As a result, I now feel much more confident in designing within Figma and making creative decisions that enhance the user experience.

In conclusion, this module has been extremely beneficial in developing both my technical and design skills. I have learned valuable tools and workflows that I can carry forward into future projects, including planning, coding, and visual design. Completing this website has given me a strong sense of achievement and has increased my confidence in my ability to design and build functional, well-structured websites from start to finish.
